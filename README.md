This is a simple todo list application, made with html, css, and JavaScript. It is designed to help you keep track of your tasks.
FEATURES:
1. It has a text input to type your tasks
2. It has a colour code to differentiate your tasks based on priority.
3. It has a progress bar to track how many of your tasks have been completed.
5. It has search task feature, a toggle for dark theme/mode.
6. Finally, it has a button to clear completed tasks.
